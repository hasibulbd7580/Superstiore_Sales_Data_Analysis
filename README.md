## Superstiore_Sales_Data_Analysis
## Overview
I have Tried to get insights from this Data set. I have created Graphs, Kpis, Dashboard to show the trends and values of the data. I have tried to turning data into valuable insights so that Business can use it.
## Dataset
The dataset contains sales transaction records, including:

- Order Details
- Product Information
- Customer Information
- Sales Revenue
- Geographic Data
- Deal Size Classification

### Key Columns
- ORDER_NUMBER
- SALES
- QUANTITY_ORDERED
- ORDER_DATE
- STATUS
- PRODUCT_LINE
- CUSTOMER_NAME
- CITY
- COUNTRY
- DEAL_SIZE

## Dashboard Features

- Deal_Size Wise Sales
- Product_Line Wise Sales
- Sales Wise Top 4 Country
- Sales Wise Bottom 4 Country
- Top 5 Cities by Sales
- Top 5 Customers by Sales
- Order_Status Wise Quantity_Ordered
- Monthly Sales Trend
- Yearly Sales Trend

## Tools Used

- Excel


## Key Insights

- Identified top-performing countries and cities.
- Analyzed product line contribution to revenue.
- Discovered sales trends across months and years.
- Examined order status distribution.
